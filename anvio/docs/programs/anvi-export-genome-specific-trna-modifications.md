This program **links tRNA-seq modification profiles to the genomes and modification enzymes responsible for them**.

It is part of the [trnaseq-workflow](../../workflows/trnaseq/) and is typically run after %(anvi-integrate-trnaseq)s has associated tRNA-seq seeds with tRNA genes in one or more genomes.

## Overview

tRNA modifications leave characteristic nucleotide substitution signatures in tRNA-seq data. This program connects those signatures to specific enzymes by:

1. Reading a user-provided %(trna-modification-enzyme-list)s that defines which genomic functions encode which modification enzymes and which canonical tRNA positions they target.
2. Searching each genome in the %(external-genomes)s file for genes annotated with those functions.
3. Using the `trna_gene_hits` table in the %(trnaseq-contigs-db)s to find tRNA-seq seeds that map to the tRNA genes in each genome.
4. Reporting modification signal (nucleotide counts and modification fractions) for each matching modification enzyme x position x seed × genome × sample combination, drawing on data pre-computed in %(modifications-txt)s and %(seeds-specific-txt)s.

The output %(genome-specific-trna-modifications-txt)s contains one row per seed per sample per matched enzyme per position the enzyme targets, for both positions where a modification was detected (SNV rows) and positions where it was not (no-SNV rows). Enzymes that target multiple canonical positions (e.g., trm61 targets both positions 57 and 58) therefore contribute one row per position. The optional `--enzyme-distribution-output` table reports the number of enzyme-encoding genes found in each genome.

## Inputs

The %(trnaseq-contigs-db)s must have been processed by %(anvi-integrate-trnaseq)s so that it contains the `trna_gene_hits` table linking seeds to genomic tRNA genes. The %(modifications-txt)s and %(seeds-specific-txt)s files are generated by %(anvi-tabulate-trnaseq)s. The genomes in %(external-genomes)s must have been annotated with gene functions (e.g., by `anvi-run-ncbi-cogs`) using the same source specified in the %(trna-modification-enzyme-list)s.

## Coverage threshold

The `--min-coverage-for-detection` parameter (default: 20) sets the minimum per-position coverage required to report modification fractions. For SNV rows, coverage is the sum of A+C+G+T reads at the modification position. For no-SNV rows, mean seed coverage is used as a proxy. Rows with coverage below the threshold have modification fractions reported as `NA`.

## Enzyme distribution

Use `--enzyme-distribution-output` to also write a table showing how many genes encoding each modification enzyme are present in each genome. This table includes every enzyme × genome combination, including genomes where the enzyme is absent (gene count = 0), making it suitable for presence/absence analysis across a collection of genomes.
